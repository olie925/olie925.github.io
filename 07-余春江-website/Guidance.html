Guidance

Ⅰ Languages

CUC_ParaConc interface supports both English and Chinese languages. Open the "CUC_Paraconc.exe.config" file in the folder with your notepad, change the value of the software interface language from "English" to "Chinese", close it and restart CUC_Paraconc, and then you will find the interface language switched to Chinese.


Ⅱ 1:1 Parallel Corpus Search

The 1:1 parallel corpus search contains English-Chinese bilingual search, alphabetic languages bilingual search and bilingual regular search, all three of which are used in exactly the same way. This website takes English-Chinese bilingual search as an example.


Ⅲ Click "English-Chinese bilingual search" to enter the interface of data loading and search parameters settings.


Ⅳ Select the source directory and saving directory

First you need to store all the source texts into one folder. Select the source folder, as well as specify the path to save the retrieved results.


Ⅴ Select data loading mode

CUC_ParaConc supports loading the aligned texts. You can store the aligned source texts and target texts into one txt file, or save them in two separate txt files. Let's take "stored in one text" as an example (if you choose "stored separately in two texts", make sure that the text prefixed are correct).


Ⅵ search scale setting

If you select "search in all texts", all files in the source directory folder will be retrieved, and if you select "search in a selection of texts", you can select the texts to be retrieved on the right side.


Ⅶ Results setting

The results can be allowed to only appear in the text box, and you can save the results into the source text. Check the boxes as appropriate.


You can now start searching. Click "search and extraction", enter the Chinese or English keyword to be searched, and click "Search". You can sort the results as needed. Click "Save results" to save the search results as a sepapate tex file for subsequent use.


Enter the line number into the text box to the right of "Search" and click "View source", you can check the context in which the sentence is located.



If you want to filter out sentences that do not contain certain words, you can enter the words in the "Please input a stoplist" box. For example, to search for sentences containing "China" but not "the People", you can enter "labor" in the stoplist and click "Search" again.
